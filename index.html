Neural Style Transfer

This is a Tensorflow.js implementation of the Neural Style Transfer described in Leon A. Gatys Et al. and much help from this TF Eager guide. The given example images from Wikipedia below are used but you can select images yourself.

Content Image

This image is the one that will be slowly transfered to aquire the style of the style image below. Therefore choose an image that you want to have a new style.

Style Image

The style of this image will be used to slowly transform the content image to this style.

Result

The result of transforming the style of the content image to the style of the style image. Transforming takes time depending on your available hardware.
